# /AgentMD Command

## Purpose

Create AGENTS.md in any project root using the universal template.

## Execution Steps

### Step 1 — Detect Project Root

Agent MUST check for project indicators (in order):

1. `.git/` directory
2. `package.json` (Node.js)
3. `pyproject.toml` or `requirements.txt` (Python)
4. `go.mod` (Go)
5. `Cargo.toml` (Rust)
6. `Gemfile` (Ruby)
7. `pom.xml` or `build.gradle` (Java)
8. Current working directory if none found

### Step 2 — Check Existing AGENTS.md

Agent MUST:

```bash
if [[ -f "AGENTS.md" ]]; then
    echo "AGENTS.md already exists"
    echo "Options:"
    echo "  1. Abort (default)"
    echo "  2. Run /AgentCheck to update instead"
    exit 0
fi
```

### Step 3 — Generate AGENTS.md

Agent MUST copy from template:

```bash
TEMPLATE="$SKILL_ROOT/skill/templates/AGENTS.template.md"
cp "$TEMPLATE" ./AGENTS.md
```

### Step 4 — Verify Structure

Agent MUST confirm file created:

```bash
if [[ ! -f "AGENTS.md" ]]; then
    echo "ERROR: Failed to create AGENTS.md"
    exit 1
fi
```

### Step 5 — Git Commit

Agent MUST commit the new file:

```bash
git add AGENTS.md
git commit -m "agentmd: bootstrap AGENTS.md

Initialize agent behavior contract with:
- Tone standards
- Documentation-first protocol
- Git discipline rules
- Quality gates (Done Criteria)
- Local checkpoint awareness
- MCP Context7 integration

Generated by /AgentMD command."
```

## Success Criteria

- [ ] Project root detected
- [ ] AGENTS.md created (or abort if exists)
- [ ] All template sections present
- [ ] Committed with descriptive message

## Error Handling

| Scenario | Action |
|----------|--------|
| Not in git repo | Warn user, proceed with creation only |
| AGENTS.md exists | Abort, suggest /AgentCheck |
| Write permission denied | Report error, suggest checking permissions |
| Git commit fails | Stage only, report to user |

## Output

On success:
```

✓ Committed: agentmd: bootstrap AGENTS.md

Next steps:

  2. Customize Decision Triggers (Section 3) for your skills
  3. Run /AgentCheck after first coding session
```
